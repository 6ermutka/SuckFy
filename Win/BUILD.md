# 🏗️ Инструкция по сборке SuckFy для Windows

## Требования

- Node.js (версия 18 или выше)
- npm (устанавливается вместе с Node.js)
- Windows 10/11

## Шаг 1: Установка зависимостей

```bash
npm install
```

## Шаг 2: Сборка приложения

```bash
npm run electron:build
```

Эта команда:
1. Соберёт React приложение (`vite build`)
2. Упакует Electron приложение с помощью `electron-builder`
3. Создаст установщик для Windows

## Шаг 3: Результат сборки

После успешной сборки файлы будут находиться в папке `dist/`:

```
dist/
├── win-unpacked/          # Распакованная версия приложения
└── SuckFy Setup 1.0.0.exe # Установщик (NSIS)
```

### Что вы получите:

- **SuckFy Setup 1.0.0.exe** - единый установщик (NSIS installer)
  - Размер: ~100-150 MB
  - При запуске установит приложение в `C:\Program Files\SuckFy\`
  - Создаст ярлык на рабочем столе и в меню "Пуск"

## Где хранятся данные пользователя

После установки и запуска приложения, все треки и метаданные сохраняются в:

```
C:\Users\<ИмяПользователя>\Documents\SuckFy\
├── Music\           # Загруженные аудио треки (.mp3, .m4a)
├── Album\           # Обложки треков (.jpg, .png)
└── library.json     # База данных (метаданные треков, плейлисты, лайки)
```

### Важно:
- Эти данные **НЕ удаляются** при удалении приложения
- Пользователь может вручную очистить кэш через Settings → Clear Cache
- Или удалить папку `Documents\SuckFy` вручную

## Конфигурация сборки

Настройки сборки находятся в `package.json`:

```json
{
  "build": {
    "appId": "com.suckfy.app",
    "productName": "SuckFy",
    "win": {
      "target": ["nsis"],
      "icon": "public/icon.ico"
    },
    "files": [
      "dist-react/**/*",
      "electron/**/*"
    ]
  }
}
```

## Добавление иконки

Если нужно добавить иконку приложения:
1. Создайте файл `public/icon.ico` (256x256 пикселей)
2. Пересоберите приложение

## Режимы запуска

### Разработка (с hot reload):
```bash
npm run electron:dev
```

### Только веб-версия (браузер):
```bash
npm run dev
```
Откроется на http://localhost:5173

### Production build:
```bash
npm run electron:build
```

## Troubleshooting

### Ошибка при сборке
Если возникает ошибка, попробуйте:
```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
npm run electron:build
```

### Приложение не запускается
Проверьте антивирус - он может блокировать неподписанное приложение

### Большой размер установщика
Это нормально! В установщик включён:
- Chromium (встроенный браузер Electron)
- Node.js runtime
- Ваше приложение

## Публикация

После сборки файл `dist/SuckFy Setup 1.0.0.exe` можно:
- Распространять напрямую (отправить пользователям)
- Загрузить на GitHub Releases
- Разместить на сайте для скачивания

**Примечание:** Для подписи приложения Windows Defender/SmartScreen нужен платный сертификат от Microsoft.
