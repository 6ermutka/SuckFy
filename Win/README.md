# 🎵 SuckFy - Windows Version

Музыкальный плеер с поддержкой SoundCloud, Spotify и локальных файлов для Windows.

![Platform](https://img.shields.io/badge/platform-Windows-blue.svg)
![License](https://img.shields.io/badge/license-ISC-green.svg)

## 🚀 Установка и запуск

### Требования
- Node.js 16+
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Режим разработки

```bash
npm run electron:dev
```

### Сборка приложения

```bash
npm run electron:build
```

## 🛠️ Технологии

- **Electron** - Desktop приложение
- **React 18** - UI фреймворк
- **TypeScript** - Типизация
- **Vite** - Сборщик
- **yt-dlp** - Загрузка треков
- **ffmpeg** - Обработка аудио

## 📝 Описание

Windows версия музыкального плеера SuckFy с полным функционалом:
- Поиск и загрузка треков из SoundCloud и Spotify
- Локальная библиотека
- Управление плейлистами
- Избранные треки
- Современный интерфейс

Оригинальная macOS версия: https://github.com/6ermutka/SuckFy
