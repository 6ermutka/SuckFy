# 🪟 Установка и сборка на Windows

## Шаг 1: Установка Node.js

1. **Скачайте Node.js:**
   - Перейдите на https://nodejs.org/
   - Скачайте **LTS версию** (рекомендуется) для Windows
   - Выберите "Windows Installer (.msi)" для вашей архитектуры (обычно 64-bit)

2. **Установите Node.js:**
   - Запустите скачанный `.msi` файл
   - Следуйте инструкциям установщика
   - ✅ Убедитесь что выбрана галочка "Automatically install necessary tools"
   - Нажмите "Install"

3. **Проверьте установку:**
   - Откройте **PowerShell** (Win + X → Windows PowerShell)
   - Введите команды:
   ```powershell
   node --version
   npm --version
   ```
   - Должны увидеть версии (например: v20.11.0 и 10.2.4)

## Шаг 2: Клонирование проекта

### Вариант A: Через Git (если установлен)
```powershell
git clone https://github.com/6ermutka/testingsuckk.git
cd testingsuckk
```

### Вариант B: Скачать ZIP
1. Перейдите на https://github.com/6ermutka/testingsuckk
2. Нажмите зелёную кнопку **Code** → **Download ZIP**
3. Распакуйте архив в любую папку
4. Откройте **PowerShell** в этой папке:
   - Shift + Правая кнопка мыши в папке → "Открыть окно PowerShell здесь"
   - Или в адресной строке проводника введите `powershell` и нажмите Enter

## Шаг 3: Установка зависимостей

В PowerShell (находясь в папке проекта):

```powershell
npm install
```

⏱️ Это займёт 2-5 минут. Будут загружены все необходимые библиотеки.

## Шаг 4: Сборка приложения

```powershell
npm run electron:build
```

⏱️ Это займёт 3-7 минут.

**Что происходит:**
1. Сборка React приложения
2. Упаковка Electron
3. Создание установщика для Windows

## Шаг 5: Результат

После успешной сборки файлы будут в папке `dist`:

```
dist/
├── win-unpacked/              # Распакованная версия
└── SuckFy Setup 1.0.0.exe     # 👈 ВАШ УСТАНОВЩИК!
```

**Размер:** ~100-150 MB (это нормально, включает Chromium и Node.js)

## Где запускать команды?

### PowerShell (рекомендуется)
1. Откройте папку проекта в проводнике
2. В адресной строке введите `powershell` → Enter
3. Вводите команды

### Или CMD (Command Prompt)
1. Win + R → введите `cmd` → Enter
2. Перейдите в папку проекта:
   ```cmd
   cd C:\путь\к\проекту\testingsuckk
   ```

### Или Windows Terminal (если установлен)
- Современный терминал от Microsoft
- Скачать: https://aka.ms/terminal
- Работает так же как PowerShell

## Запуск в режиме разработки

Если хотите протестировать перед сборкой:

```powershell
npm run electron:dev
```

Откроется приложение в режиме разработки с hot reload.

## Возможные проблемы

### ❌ "npm не является внутренней командой..."
**Решение:** Node.js не установлен или не добавлен в PATH
- Переустановите Node.js
- Перезагрузите PowerShell/компьютер

### ❌ Ошибка при установке зависимостей
**Решение:**
```powershell
npm cache clean --force
npm install
```

### ❌ Антивирус блокирует установщик
**Решение:** Это нормально для неподписанных приложений
- Добавьте в исключения антивируса
- Или временно отключите при сборке

### ❌ Недостаточно места на диске
**Требуется:** минимум 2 GB свободного места

## Куда устанавливается приложение?

После запуска установщика:
- **Программа:** `C:\Program Files\SuckFy\`
- **Данные пользователя:** `C:\Users\<ИмяПользователя>\Documents\SuckFy\`
  - Music\ - скачанные треки
  - Album\ - обложки
  - library.json - база данных
